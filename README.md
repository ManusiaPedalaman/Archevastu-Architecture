## 🏗️ Tentang Project

**Archevastu** hadir sebagai ruang pamer digital untuk perpaduan antara seni arsitektur dan interaksi antarmuka tingkat tinggi. Kami tidak sekadar menampilkan deretan foto bangunan; kami menyajikan sebuah **pengalaman imersif** melalui layar Anda.

Project ini lahir dari visi untuk mendobrak batasan presentasi karya desain arsitektural tradisional berformat statis. Dibangun secara saksama dengan komitmen pada **Estetika**, **Animasi Berperforma Tinggi**, dan **Seni Bercerita Visual (Scrollytelling)**. Archevastu mengartikulasikan setiap lekuk desain arsitektur menjadi pengalaman interaktif yang dinamis dan berkesan.

### ✨ Fitur Utama

*   **🎬 Scrollytelling Immersive:** Pengalaman bercerita responsif di mana visualisasi gedung, denah, dan deskripsi proyek merespon secara dinamis pada tiap guliran (*scroll*) jemari Anda.
*   **🎭 Animasi Canvas yang Hidup:** Menghadirkan transisi visual yang halus serta elemen arsitektur dinamis dengan kekuatan animasi WebGL/Canvas di balik layar.
*   **🌓 Tema Gelap/Terang Memukau:** Nikmati eksplorasi karya dalam kondisi cahaya optimal. Mendukung pengalaman *Dark Mode* kustom yang premium tanpa mengurangi detail desain.
*   **⚡ Navigasi Selembut Sutra:** Diperkuat oleh utilitas *Smooth Scroll* terbaru yang membuat perjalanan Anda antar galeri desain murni mengalir tanpa jeda.
*   **📐 Fokus pada Estetika Ruang:** Setiap elemen tipografi minimalis dan *grid* dirancang agar sorotan utama tetap bertumpu pada mahakarya arsitektur itu sendiri.

## 🛠️ Tech Stack

Dapur project eksperiensial super cepat ini dibangun menggunakan gabungan ekosistem dan teknologi web termodern tahun ini:

*   **Core:** [Next.js](https://nextjs.org/) (App Router) & [React 19](https://react.dev/)
*   **Styling Utama:** [Tailwind CSS v4](https://tailwindcss.com/) (Sistem utilitas terkini untuk menghadirkan Dark/Light Theme yang memukau)
*   **Animasi Dinamis:** [Framer Motion](https://www.framer.com/motion/) yang memastikan *scrollytelling* dan *micro-actions* berjalan super mulus
*   **Pengalaman Skroling:** [Lenis](https://lenis.darkroom.engineering/) (Utilitas Smooth Scroll UI untuk sensasi navigasi premium)
*   **Ikonografi:** [Lucide React](https://lucide.dev/)
